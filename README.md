# 중외정보기술 Server Side 템플릿

## 🖥️ 소개
중외정보기술 프로젝트 수행시 빠른 구축을 위한 서버 템플릿

## ⚙️ 개발 환경
Java : JDK 1.8+

Tomcat : 8.0+

Framework : spring + egovFramework

Build : Maven

Mapper : Mybatis

### 프로젝트 구조

```bash
+-- java # 서버 리소스
|   +-- egovframework
|     +-- cmm
|     +-- cmmn
|     +-- index
|     +-- interceptor
|     +-- member
|     +-- scheduler
|     +-- sec
+-- resource # spring bean 등록을 위한 XML
|   +-- egovframework
|      +-- spring
+-- READMD.md
```
